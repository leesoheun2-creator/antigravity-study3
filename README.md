# 고려대학교 GPA 피드백 매니저
## 이 프로그램은 고려대학교 학생을 위한 학점 피드백 매니저입니다.

**고려대학교** 학생을 위한 학점 피드백 매니저 작성법
- 시작 때마다 사용자의 학번, 이름, 비밀번호를 입력받기
- 학기를 선택하기 (1학년 1학기~4학년 2학기 중 선택)
- 모든 수강한 과목에 대하여, 수강한 과목+해당과목이 전공필수/전공선택/교양필수/교양선택 중에 무엇인지
- p/np 과목인지 묻고, 맞다면 Pass와 Nonpass 중에 무엇인지, 아니라면 A플러스와 F까지 중에 해당 학점이 몇학점인지의 3가지를 입력받기
- 목표 GPA를 물어보기 (4.5 학점 체계 기준)

- GPA 피드백 매니저의 할일
- [ ] 학점 산출하기 (단순평균이 아니라 학점의 가중치에 따라 다르게 산출)
- [ ] 학점 분석하기 1 - 총 학점 중 전공필수/전공선택 과목을 모은 '전공 학점', 교양필수/교양선택 과목을 모든 '교양 학점'을 따로 산출하기
- [ ] 학점 분석하기 2 - 목표점수가 되기까지 개선할 점을 피드백
만약 학점이 목표보다 많이 낮다 → 학점을 따기 쉬운 꿀교양을 더 들어라+고려대학교의 꿀교양을 찾아서 추천하기
전공,교양 중 한 쪽에 지나치게 치우쳐 있다 → 나머지 것을 보충하라고 이야기해주기
고려대학교의 교양필수 과목을 모두 듣지 않은 상태이다 → 더 들어야 할 필수 과목을 명시
고려대학교의 전공필수 과목을 모두 듣지 않은 상태이다 → 더 들어야 할 필수 과목을 명시
특정 과목(ex.영어, 심리학)을 잘 못한다 → 다음 학기부터는 피해서 선택하라고 조언
- [ ] 졸업 완성도 계산하기 1 - 이 학생이 총 몇학점을 들었는지 계산하고, 진행률 %도 함께 표시하기
- [ ] 졸업 완성도 계산하기 2 - 영어 성적과 한자 성적 완성 여부를 조사받고, 종합적으로 판단해서 몇 년 정도 걸릴지 계산하기

- 언제? : 학기말 성적 공시 시점
